Esta página apresenta os recursos básicos do editor. Os exemplos podem ser copiados e colados diretamente nos campos de texto do sistema.

Como funciona

O editor aceita marcações simples para formatar o texto. Você pode digitar manualmente ou usar os botões da barra de ferramentas.

Para fórmulas matemáticas, use sempre a expressão entre $...$.

1. Negrito

Use dois asteriscos antes e depois do trecho.

Exemplo para copiar
**Trecho em negrito**
Resultado esperado
Trecho em negrito

2. Itálico

Use um asterisco antes e depois do trecho.

Exemplo para copiar
*Trecho em itálico*
Resultado esperado
Trecho em itálico

3. Lista com marcadores

Inicie cada linha com -.

Exemplo para copiar
- marca de frenagem
- dano no para-choque
- posição final do veículo
Resultado esperado
  • marca de frenagem
  • dano no para-choque
  • posição final do veículo

4. Lista numerada

Inicie cada linha com 1., 2., 3. e assim por diante.

Exemplo para copiar
1. observar o vestígio
2. medir o comprimento
3. aplicar a fórmula
Resultado esperado
  1. observar o vestígio
  2. medir o comprimento
  3. aplicar a fórmula

5. Fórmulas matemáticas inline

Para escrever fórmulas, coloque a expressão entre $ e $.

Use apenas fórmulas inline, no formato $...$.
Exemplo simples
A energia cinética é dada por $E_c = \frac{1}{2}mv^2$.
Exemplo com potência
$x^2 = y^3$
Exemplo com índice
$v_0^2 = v^2 - 2a\Delta s$
Exemplo com raiz
$x = \sqrt{a^2 + b^2}$
Exemplo com fração
$v = \frac{d}{t}$

6. Exemplo aplicado: velocidade a partir de marca de frenagem

A fórmula abaixo pode ser usada em situações em que se deseja expressar a velocidade estimada em função da marca de frenagem, conforme o modelo adotado no texto técnico.

Fórmula para copiar
$v = \sqrt{2 \mu g d}$
Frase completa para copiar
A velocidade estimada pode ser expressa por $v = \sqrt{2 \mu g d}$, em que $\mu$ representa o coeficiente de atrito, $g$ a aceleração da gravidade e $d$ o comprimento da marca de frenagem.

7. Exemplo aplicado: equação de Torricelli

Outro exemplo útil em textos técnicos envolvendo movimento é a equação de Torricelli.

Fórmula para copiar
$v^2 = v_0^2 + 2a\Delta s$
Frase completa para copiar
A relação entre velocidade e deslocamento pode ser expressa por $v^2 = v_0^2 + 2a\Delta s$.

8. Observações

  • Use **texto** para negrito.
  • Use *texto* para itálico.
  • Use - para listas com marcadores.
  • Use 1. para listas numeradas.
  • Use $...$ para fórmulas matemáticas inline.
